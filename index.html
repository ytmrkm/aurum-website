aurumOS

is a powerful, user-friendly operating system based on Void Linux. It offers a seamless and efficient experience with two editions tailored for different needs: the Standard Edition with GNOME 46 and the Lite Edition with Budgie. Preloaded with essential applications and the nix package manager, aurumOS ensures a robust and versatile environment for all users.

Good base

aurumOS is built from scratch on Void Linux, ensuring a unique and independent development path without relying on other distributions.

Uses Runit

Our operating system utilizes the lightweight and fast Runit init system, providing quick boot times and efficient process management.

Seamless Installation

With the "system-installer," setting up aurumOS is a breeze. Enjoy a straightforward and user-friendly installation process.

Enhanced Performance

Experience smooth and responsive performance with our optimized GNOME 46 and Budgie environments, ensuring you get the best out of your hardware.

XBPS and Nix

Preinstalled with the XBPS, and Nix package managers, aurumOS offers powerful and flexible package management, making software installation and updates a breeze.

Essential Apps Included

aurumOS comes with Firefox, VLC, and LibreOffice preloaded, so you can start working, browsing, and enjoying media right out of the box.
